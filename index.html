As cybersecurity threats continue to rise, a major contributing factor is poor passwords among regular internet users including weak, reused, and compromised passwords which can lead to harmful data breaches. Poor password practices like weak and reused passwords are a major cause of cybersecurity vulnerabilities and data breaches. The research question is: can a fun, game-like web application help promote better password habits among internet users? This study will develop password checking web apps using gameplay elements to see if making security education interactive improves the adoption of strong, unique passwords.
